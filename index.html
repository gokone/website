About Project

Background

It started with an idea that is shared here: https://gist.github.com/prayank23/6f54e9a27f057abd1182436e7f88d1ac and has been extended to include a few more features.

What is it?

A Decentralized application for betting on cricket matches.

Why?

There are lot of websites which can be used to bet on cricket matches using bitcoin, however you need to trust the website. This application is built on top of the bitcoin and is not reliant on any third party website. Users donâ€™t need to deposit any money to use the application as they will always hold the keys.

How does it work?

An application which is still in alpha stage would be available to bet on cricket matches. Discreet Log Contracts are used for oracles. Orderbook is maintained in the application so that users can look at all the bets being placed. Orderbook works based on mempool, transactions related to bitcoin addresses generated in the application and GNUnet API will be used later for most of the things.
